Plan Topographique Vierge - Visualisation et Export

Instructions :

Ce plan topographique vierge peut être utilisé comme base pour vos projets d'aménagement ou vos esquisses. Vous pouvez :

  1. Visualiser le plan directement dans cette page
  2. Télécharger le fichier SVG pour l'éditer dans des logiciels comme Inkscape, Adobe Illustrator, etc.
  3. Générer un PDF à partir de cette page en utilisant la fonction d'impression de votre navigateur
Télécharger le SVG

Pour générer un PDF :

  1. Cliquez sur le bouton "Imprimer / Générer PDF"
  2. Dans la boîte de dialogue d'impression, choisissez "Enregistrer au format PDF" ou "Microsoft Print to PDF" (selon votre système)
  3. Choisissez un nom de fichier et enregistrez

Note : Pour une meilleure qualité, assurez-vous que l'option "Échelle" est réglée sur 100% dans les paramètres d'impression.
